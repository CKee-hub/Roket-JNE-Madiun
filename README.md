# 🛵 MadiunEats

Platform kuliner lokal Madiun — pesan makanan langsung via WhatsApp.

## Cara Deploy ke Vercel

1. Upload folder ini ke GitHub
2. Login ke vercel.com
3. Import repository
4. Klik Deploy — selesai!

## Cara Jalankan Lokal

```bash
npm install
npm start
```
